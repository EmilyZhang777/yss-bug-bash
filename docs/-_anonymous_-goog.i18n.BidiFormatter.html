unicodeWrap(str, placeholderopt) → {string}

Formats a string of unknown directionality for use in plain-text output of the context directionality, so an opposite-directionality string is neither garbled nor garbles what follows it. As opposed to #spanWrap, this makes use of unicode BiDi formatting characters. In HTML, its *only* valid use is inside of elements that do not allow mark-up, e.g. an 'option' tag.
